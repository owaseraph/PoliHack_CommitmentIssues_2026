Installation guide

Download the extension package above and save the .zip file.
STEP 02

STEP 03
Click Load unpacked and select the extracted extension folder.
STEP 04
Open Gmail. The PhishGuard shield icon appears — click it and log in to activate scanning.
FAQ
What permissions does the extension need?
The extension requests access to mail.google.com only. It reads email metadata visible in the Gmail interface and sends it to the PhishGuard API for analysis. No other sites are accessed.
Does the extension store my emails?
No. Emails are sent to the local PhishGuard server for scanning and immediately discarded. Nothing is stored in the extension or sent to third parties beyond Google's own APIs.
Does it work without the web dashboard?
The extension connects to the same local PhishGuard backend. You need the server running on localhost:8000 for real-time scanning to work.
Can I scan a specific email, not just the latest ones?
Yes — open any email in Gmail and click the PhishGuard icon in the toolbar to scan that specific message on demand.
